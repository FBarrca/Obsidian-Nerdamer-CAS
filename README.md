# Nerdamer Calculator

Do powerful math calculations inline with your notes, without loosing focus.

## ⚙️ Features

- Add variables

`x:=1`

- Solve equations
`solve (x+10=z,z)`

- Evaluate expressions
`x=?`

- Create functions
`f(x):=x^2`

- Evaluate functions
`f(2)=?`

You can use also commands such as diff or integral

`a:=2`
`f(x):=x^a - a*x+b`
`diff(f(x),x)=?`
